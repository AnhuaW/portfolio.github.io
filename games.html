Here are some game projects I worked on for the Game Development course EECS 494 @ Umich

Zelda Remake

A Zelda remake built with Unity, features a costum level with a new harp weapon and brand new enemies.

  • Implemented weapons and the switching weapons mechanic
  • Implemented Camera movement when entering/exiting rooms
  • Implemented the Inventory system and collectibles
  • Implemented the Locked doors mechanic
  • Implemented the music, SFX and UI
  • Learn more

AstroEscape

A 2D puzzler features gravity control in space

  • Designed and coded the game with C# in Unity from scratch in two weeks
  • Implemented gravity control using the Unity Physics 2D system
  • Experimented with spawning maps as game objects so that player status can carry over levels
  • Learn more

RoBoo

A pixel-style 2D platform puzzler built from scratch.

  • Implemented character movement and animation
